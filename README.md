## Funcionamiento del proyecto: 
El proyecto simula el funcionamiento de una página web dirigida a la busqueda de tutores en las áreas de matemáticas, física y programación. En este las persona que se encuentran interesadas podrán realizar busquedas mediante filtros según sus necesidades y reservar sus clases.
